Batman Re-soundtracked

As part of Audio 2 module, I re-soundtracked Batman Arkham Asylum. This is a short video clip of one of the recent Batman games. The original soundtrack has been stripped away and replaced with my own interpretation of what I think the batman game should sound like. The audio was recorded using a Zoom H4n recorder and mixed using Pro-Tools 8. The background music was composed by me on my piano and then inputted using a MIDI keyboard into Pro-Tools. As in the other videos no copyright infringement is intended, this is merely a sample of work from my Audio2 module from university. The content is only used to show that I can design, record and mix my own samples and use these skills in a real-life scenario.

Processing.org Music Visualisation

This is a music visualiser coded in Processing. Each of the elements on screen are responding directly from a wav file input. The code was also appended to cater for live visuals in a concert capacity.
